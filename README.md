## Data Science Projects

**1. A/B Testing and Chi-Square Analysis**
Statistical hypothesis testing to evaluate whether two variants produce 
significantly different outcomes — core experimental design methodology.

**2. House Price Prediction — Linear Regression (99.6% accuracy)**
Predictive regression model for housing prices using feature engineering 
and model evaluation.

**3. Country Clustering — PCA and K-Means**
Unsupervised dimensionality reduction and clustering to group countries 
by development indicators.

**4. Email Spam Detection — Naive Bayes, Random Forest, LSTM (97% accuracy)**
Text classification comparing traditional ML classifiers against a deep 
learning LSTM sequence model.

**5. Brain Tumor Detection — CNN, VGG19, InceptionV3 (100% accuracy)**
Medical image classification using convolutional neural networks and 
transfer learning on MRI scan data.

**6. Walmart Sales Forecasting — ARIMA Time Series**
Time series decomposition and ARIMA forecasting for retail demand prediction.

**7. Rainfall Prediction — Logistic Regression, KNN, Random Forest, Decision Tree**
Binary classification comparison across four algorithms for weather 
prediction in Australia.
